## 프로젝트별 실행 방법

### cpp_project (C++ 프로젝트)
**컴파일:**
```bash
cd cpp_project
g++ main.cpp -o main
```

**실행:**
```bash
./main
```

### python_project (python 프로젝트)
**실행 및 컴파일:**
```
python3 main.py
```
