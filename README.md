# 숭실대학교(SSU) 디지털미디어학과 개인 저장소

## 📝 소개
이 저장소는 숭실대학교 디지털미디어학과 수업에서 다루는 코드, 프로젝트, 과제 등을 백업하고 관리하기 위한 공간입니다.

## 📚 목적
- 수업 내용 복습 및 정리
- 과제 및 프로젝트 버전 관리
- 개인 포트폴리오 자료 축적

## 내용
- 프로그래밍 2 : C++ 및 WASM
- 미디어앤테크 : p5.js
- AI와 데이터기초 : python

## 프로젝트별 실행 방법

### cpp_project (C++ 프로젝트)
**컴파일:**
```bash
cd cpp_project
g++ main.cpp math_utils.cpp -o main
```

**실행:**
```bash
./main
```

**기능:** 수학 유틸리티 함수(add)를 사용한 기본 C++ 프로그램
